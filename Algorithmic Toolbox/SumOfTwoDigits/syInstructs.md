## Sum of Two Digits

#### Input format: 
    Integers a and b on the same line(separated by space)

#### Output format:
    The sum of a & b

#### Constraints:
    0 <= a,b <= 9.

#### Time limit :
    
    C       C++     Java        Python      JavaScript
    1        1       1.5           5             5
    
####  Memory limit:
    512Mb
